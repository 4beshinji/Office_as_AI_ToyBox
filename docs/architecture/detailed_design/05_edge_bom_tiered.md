## コスト削減 Tips

- **MH-Z19C**: AliExpress で ¥1,500〜2,000 で入手可能（国内の半額）
- **XIAO ESP32-C3**: Tier 1 で最もコスパが良い（¥900）
- **DHT22 vs SHT31**: DHT22 (¥600) は精度がやや劣るが Tier 1 には十分
- **PIR**: AM312 (ミニサイズ ¥200) は HC-SR501 の代替として小型化に有効
